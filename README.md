## Install
1. Click [release](https://github.com/ltfred/pgl/releases) download latest package.
2. `pip install <package_name>`

## Use

You can use `pgl --help` to get help.
```
Commands:
  clone    Clone project
  config   Edit config file
  sync     Sync projects
  version  Display version for pgl
```
